About

Is a strategic game written by

Yuriy Stanchev as part of Master Thesis at New Bulgarian University, Sofia, Bulgaria, 2012. You can read more about the Author of the Game and other projects on his personal page:

http://stanchev.netsecl.com

The modified version with six players was done in Velbazhd Software LLC, Sofia, Bulgaria, 2015. This version of the game can be found at: http://github.com/TodorBalabanov/ColorsOverflow

Any suggestions for the Game improvement, bug reports etc. are welcome. You can contact the author: i_stanchev at ml1.net
