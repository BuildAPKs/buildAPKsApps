Make DuckDuckGo your default search engine

First, try this:
Navigate to the home screen, press the search button, type foo and press Enter
Check "Use by default for this action." and press Duck Duck Go. Done!

Didn't work?
This is usually because Google is set as the default web search engine.
To rectify this, go to Settings > Applications > Manage Applications. Look for "Google Search" or "Google (enhanced search)". Select it and choose the "Clear defaults" button. Then redo the procedure above.

No suggestions showing up?

Navigate to Settings > Search > Searchable Items and check Duck Duck Go.

Using Android 2.2? Want a DuckDuckGo widget on your homescreen?

First, navigate to Settings > Search > Searchable Items and check Duck Duck Go.
Then, go to your home screen and long press. Choose Widgets > Google Search > Duck Duck Go. Done! Enjoy :-)
